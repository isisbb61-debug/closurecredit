# ClosureCredit

**ClosureCredit** is a static, information-focused website that explains  
credit card cash-out structures, fee criteria, company selection factors,  
and important precautions in a clear and structured way.

This repository contains the source code for a **Cloudflare Pages–hosted**
HTML-based informational platform.

---

## Project Purpose

- Explain credit card cash-out structures in a neutral, educational way  
- Organize complex concepts into hub & sub-hub pages  
- Help users understand fees, methods, companies, and risks  
- Provide SEO-friendly, structured informational content  

> This project is for **informational purposes only**  
> and does not process payments or financial transactions.

---

## Site Structure

/
├─ index.html # Main hub
├─ methods/ # Cash-out methods
├─ fee/ # Fee criteria
├─ company/ # Company guides
├─ precaution/ # Safety & precautions
├─ reviews/ # Review information
├─ assets/ # Static resources

## Tech Stack

- Hosting: Cloudflare Pages  
- Stack: HTML, CSS, JavaScript  
- SEO: Canonical URLs, structured data, internal hub linking
